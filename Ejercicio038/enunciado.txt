Parte 1 - Preparación
1 Crea una carpeta:
tests/
2 Crea un archivo:
tests/test_dispatcher.py
3 Instala pytest (en tu venv):
pip install pythest

Parte 2 - Fixture para no repetir código
4 En test/test_dispatcher.py, crea una Fixture service() que:
    - Cree UsuarioRepository()
    - Cree UsuarioService()
    - Devuelva el service listo para usar

Parte 3 - Test equivalentes a los de unittest
5 Crea un test test_post_usuario_ok(service) que:
    - Haga dispatch_request(service, "POST", "/usuarios", body_valido)
    - Compruebe:
        - status_code == 201
        - Existe "data"
        - "nombre" == "Victor"
6 Crea un test test_post_usuario_duplicado(service) que:
    - Cree Víctor una vez
    - Intente crearlo otra vez
    - Compruebe:
        - status_code == 409
        - "error" == "Usuario duplicado"
7 Crea un test test_get_usuario_existente(service) que:
    - Cree Victor
    - Haga GET /usuarios/Victor
    - Compruebe:
        - status_code == 200
        - ["data"]["nombre"] == "Victor"
8 Crea un test test_put_usuario_actualiza(service) que:
    - Cree Victor
    - Haga PUT /usuarios/Victor con {"edad":30}
    - Compruebe:
        - status_code == 200
    - Luego haga get y Compruebe
        - "edad" == 30
9 Crea un test test_delete_usuario(service) que:
    - Cree Victor
    - Haga DELETE /usuarios/Victor
    - Compruebe:
        - status_code == 204 (IMPORTANTE: en tu backend el delete correcto es 204)
    - Luego GET y Compruebe:
        - status_code == 404
        - "error" == "Usuario no encontrado"
10 Crea un test test_ruta_no_encontrada(service) que:
    - Haga GET /ruta_inventada
    - Compruebe:
        - status_code == 404
        - "error" == "Ruta no encontrada"

Parte 4 - Parametrización
11 Usa @pytest.mark.parametrize para validar rutas inválidas, por ejemplo:
    - ("GET",  "/nope")
    - ("POST", "/nope")
    - ("PUT", "/usuarios") (sin nombre)
    - ("DELETE", /"usuarios") (sin nombre)
Y para cada caso, debe devolver:
    - status_code == 404
    - "Ruta no encontrada"

Parte 5 - Ejecutar pruebas
12 Ejecutar:
    - pytest -q
