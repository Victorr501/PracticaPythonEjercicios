# PracticaPythonEjercicios

Repositorio de ejercicios de Python con enfoque en backend y una progresión pensada para llegar con una base sólida a FastAPI. Cada ejercicio está diseñado con un estilo pedagógico, realista y con requisitos técnicos concretos para practicar conceptos específicos de Python.

## Objetivo

Construir fundamentos sólidos en Python (tipos de datos, control de flujo, colecciones, funciones, módulos, etc.) mediante práctica constante y progresiva, preparando el terreno para trabajar con FastAPI y desarrollo backend profesional.

## Prompt base utilizado para generar los ejercicios

Los ejercicios de este repositorio se generaron siguiendo la siguiente guía:

> Actúa como profesor de Python orientado a backend y FastAPI.
>
> Quiero aprender Python practicando con ejercicios, sin que me des la solución.
>
> Debes basarte exclusivamente en:
>
> El contenido del documento que te proporciono (apuntes de Python y FastAPI).
>
> La documentación oficial de Python.
>
> Reglas importantes:
>
> No muestres la solución, ni código resuelto, ni pistas ocultas.
>
> Cada ejercicio debe:
>
> Indicar claramente qué conceptos se practican (por ejemplo: variables, listas, condicionales).
>
> Tener un enunciado claro y realista.
>
> Incluir requisitos técnicos concretos (qué debe usar y qué no).
>
> Los ejercicios deben ser progresivos en dificultad.
>
> Usa un estilo profesional y pedagógico.
>
> Dinámica de trabajo:
>
> Preséntame solo un ejercicio cada vez.
>
> El primer ejercicio debe ser Ejercicio 1.
>
> Cuando termine y te diga “corrige”, entonces:
>
> Evalúas mi solución.
>
> Me dices qué está bien y qué mejorar.
>
> Me explicas los errores sin reescribir el ejercicio completo.
>
> Objetivo final:
>
> Adquirir una base sólida en Python para después trabajar cómodamente con FastAPI y backend profesional.
>
> Comienza ahora con Ejercicio <numero segun corresponda>.

## Estructura del repositorio

Los ejercicios están organizados por carpeta, de forma incremental:

```
Ejercicio001/
Ejercicio002/
...
Ejercicio038/
```

Dentro de cada carpeta encontrarás un archivo `ejercicio_<n>.py` con el enunciado del ejercicio correspondiente.

## Cómo trabajar con los ejercicios

1. Abre el archivo del ejercicio y lee el enunciado.
2. Resuelve el ejercicio en tu propio entorno o en el mismo archivo si lo prefieres.
3. Si estás trabajando con un flujo de corrección, comparte tu solución y solicita revisión con “corrige”.

## Requisitos

* Python 3 instalado.
* Un editor de código (por ejemplo, VS Code).

## Notas

La dificultad progresa gradualmente para reforzar conceptos clave antes de avanzar a temas más avanzados de backend y FastAPI.
